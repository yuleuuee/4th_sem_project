*************  Home Page (主页) ： ****************
**************************************************


1. 在主页的右上角有一个按钮，当我们点击它时，页面的背景颜色会随着按钮内的图标而变化。

2. 我们可以改变3种背景颜色； 白天时间，黄昏时间和晚上时间。

3. 我还放了一个点击按钮的声音。



************* CONTENT PAGE (内容页) ：**************
***************************************************


1. 在左上角有一个灯泡，点击后会将背景颜色更改为夜景，反之亦然 

2.每季照片下方都有一个简短的文字，还有一个显示更多按钮，点击它可以看到完整的文字。

3.每张图片文字下方的 ll 心用于给图片点赞，它会保存图片获得的点赞数。点赞时的声音很好听.

4.当我们点击评论图标时，我们可以写评论并发表。 它会出现在每张图片的下方。

5.我们也可以通过点击要删除的评论来删除评论.

6.此处的分享按钮仅用于删除评论输入框。

7.在此页面的底部有一个向上滚动按钮； 当我们点击它时，它会将页面滚动到顶部并且过程很顺利.



************* MEDIA PAGE(媒体页) ：****************
***************************************************


1.有和以前一样的灯泡。

广告部分：

2.当我们刷新页面有一个倒计时，6秒后会出现广告.

3.并且我们只能在十字图标的颜色从灰色变为红色的5秒后删除广告。（只能删除第二个广告）

4.第二个广告有一个由“mouseover”和“mouseout”事件完成的不透明度变化过程。


抖音部分：
5.我们可以看到抖音图标在跳舞，这是“setTimeout”和“setInterval”的结果.

6.我们可以在“我设计的抖音”上发表评论，当我们按下回车键时，评论就会被发布。它会在几秒钟后消失.

7.当我们继续点击“给予喜欢”按钮时，就会出现爱心，其他表情符号会飞起来.

第三部分分：

8.如果我们按键盘上的“Shift + Z”键，那么我们可以全屏查看图像.

9.我们只需单击图像下方的打印按钮即可打印图像。

10.我们可以通过选择一个“全选按钮”来选择所有项目，反之亦然




************* GALLERY (画廊) ：****************
***********************************************


1.有和以前一样的灯泡。

2.在这个页面由滑块组成，我们可以慢慢拖动图片，呈现出照片的 3D 效果。（ 就是 Swiper ）

3.包含一个饼图，这是从互联网上获取的，我只是更改了一些数据和边界半径。（ 就是 ECharts ）

Ajax 部分：

4.使用北京的“Ajax”当前日期和时间。

5.我们可以从“https://api.github.com/users”网站获取一些假用户的照片、id和登录id



************* MAGIC PAGE (魔法页面) ：***************
***************************************************


1.表单验证存在，这里用户名不能为空，输入值不能少于4个字母。

2.当用户名密码不匹配时，将出现警告消息。我们可以点击红叉标记关闭警告框。

3.也有密码看和隐藏的眼睛

4.我们可以通过点击忘记密码按钮获取实际的用户名和密码.

5.编写正确的输入和密码，将在其中打开一个新的输入框，我们必须输入任何颜色名称或代码。

6.然后神奇的事情发生了，页面背景颜色会根据我们输入的值而改变(颜色名称或颜色代码必须正确，否则将不起作用).
